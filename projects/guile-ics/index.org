* Guile-ICS

[[https://www.gnu.org/software/guile/][GNU Guile]] iCalendar ([[https://tools.ietf.org/html/rfc5545][RFC 5545]]) and vCard ([[https://datatracker.ietf.org/doc/html/rfc6350][RFC 6350]]) parser for GNU Guile.

** Features

- Reading and writing iCalendar and vCard formats.
- Conversion between formats.
- CLI tool that provides convenient UI for the library capabilities.

** Source Code Repository

[[https://github.com/artyom-poptsov/guile-ics]]

** Documentation

You can read the documentation online [[./manual/][here]] or use =info guile-ics= on your
GNU/Linux distribution with Guile-ICS installed.
